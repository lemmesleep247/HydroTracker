## Supported Languages

HydroTracker currently ships with English as the source language. The following target languages are active on Crowdin:

- Chinese (Simplified)
- French
- German
- Portuguese (Portugal)
- Spanish
- Turkish

## Requesting a New Language

If you would like to translate HydroTracker into a language that is not listed, please open a [GitHub Discussion](https://github.com/Econ01/HydroTracker/discussions). Only project managers can add new languages in Crowdin.
